# Blinkit Sales Analysis Dashboard

## Project Overview

This project analyzes **Blinkit grocery sales data** to uncover insights related to sales performance, product categories, and outlet distribution.
The analysis was performed using **SQL for data exploration and querying**, and **Power BI for building an interactive dashboard** to visualize key business metrics.

## Tools & Technologies

* **SQL** – Data querying and analysis
* **Power BI** – Dashboard creation and data visualization
* **Dataset** – Blinkit sales dataset (8,000+ records)

## Objectives

* Analyze sales trends and customer purchasing behavior
* Identify top-performing product categories and outlets
* Calculate important business KPIs
* Create an interactive dashboard for data-driven decision making

## Key KPIs Analyzed

* Total Sales
* Average Sales
* Number of Items
* Average Ratings
* Sales by Outlet Establishment
* Sales by Outlet Type
* Sales by Outlet Size
* Sales by Outlet Location

## SQL Analysis

SQL queries were used to perform:

* **Data aggregation** to calculate total and average sales
* **Category-wise and outlet-wise analysis**
* **Filtering and grouping** to identify sales trends
* **Joins and conditional queries** to derive business insights

Example SQL queries included:

* Total sales by product category
* Outlet performance analysis
* Location-based sales distribution
* Average rating by product category

## Power BI Dashboard

An **interactive Power BI dashboard** was developed to visualize insights from the dataset.

Dashboard features include:

* Sales distribution by **product category**
* Performance by **outlet size and outlet type**
* **Location-wise sales analysis**
* KPI cards for **Total Sales, Average Sales, No. of Items and Ratings**
* **Filters and slicers** for interactive data exploration

## Key Insights

* Certain product categories contribute significantly to overall sales.
* Outlet size and type have a strong impact on sales performance.
* Customer ratings vary across product categories.

## Project Outcome

This project demonstrates the use of **SQL for data analysis** and **Power BI for business intelligence dashboards**, helping transform raw data into meaningful insights for decision-making.
